    Sensoriamento automático para irrigação de lavouras com IoT

    Este projeto é uma tarefa universitária para o curso "Análise e Desenvolvimento de Software (ADS)",no qual se baseia no pilar de sustentabilidade ambiental na implementação de tecnologias de IoT de sensoriamento automático para monitorar e controlar o processo de irrigação de forma mais precisa e eficiente.     Gestão de resíduos sólidos da cidade de São Sebastião com Power BI

    Este projeto é uma tarefa universitária para o curso "Análise e Desenvolvimento de Software (ADS)". O objetivo do projeto foi desenvolver uma ferramenta ágil, simples e acessível que possibilita uma maior praticidade na gestão de resíduos sólidos da cidade de São Sebastião, localizada no litoral norte de São Paulo. Foi confeccionado um Dashboard em Power BI com os dados municipais da coleta de lixo orgânico e reciclável da cidade, com horários, dias e quantidade de resíduos produzido por zona de coleta. O projeto tem como intuito informar o cidadão e conscientiza-lo da produção excessiva de lixo.

  • Power BI;
  • Kanban;

    Rede Interna de Servidores

    Este projeto é uma tarefa universitária para o curso "Análise e Desenvolvimento de Software (ADS)". Neste trabalho, apresentamos uma simulação do funcionamento de uma rede de computadores dentro da Hidrel, demonstrando a aplicação prática de um sistema de roteamento dinâmico. Através dessa simulação, exploramos como os protocolos de roteamento dinâmico podem otimizar a performance e a resiliência da rede, adaptando-se às mudanças e garantindo a continuidade dos serviços oferecidos pela empresa.

    Esse projeto foi desenvolvido com as seguintes ferramentas:

  • CISCO PACKET TRACER;
  • Endereçamento IP;
  • Gestão de infraestrutura de TI;
  • Governança de TI.
